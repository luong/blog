# 12G Dev Blog

This website is built using [Docusaurus 3](https://docusaurus.io), a modern static website generator. Posts are created in [Markdown](https://www.markdownguide.org) format.

This place is for sharing tips and news of software developments. Categories are various topics: nodejs, golang, javascript, php, etc.

[See the live site](https://12g.co) 

***

### Installation

```
$ npm install
```

### Local Development

```
$ npm start
```

### Build

```
$ npm run build
```

### Deployment

Currently this project is being deployed on [Vercel](https://vercel.com).
